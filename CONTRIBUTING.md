# Contributing to gitoma-bench-ladder

## How to Contribute

1. Fork the repository on GitHub.
2. Clone your forked repository to your local machine:

   ```bash
   git clone https://github.com/YOUR-USERNAME/gitoma-bench-ladder.git
   ```
3. Create a new branch for your feature or bugfix:

   ```bash
   git checkout -b feature-name
   ```
4. Make your changes and ensure the code follows PEP8 guidelines.
5. Run tests to verify your changes:

   ```bash
   python -m pytest tests/
   ```
6. Commit your changes with a clear and concise message.
7. Push your branch to your forked repository:

   ```bash
   git push origin feature-name
   ```
8. Open a pull request from your branch to the main repository.
