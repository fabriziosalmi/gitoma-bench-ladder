# Contributing to gitoma-bench-ladder

## Getting Started

1. Fork the repository on GitHub.
2. Clone your forked repository to your local machine:

   ```bash
   git clone https://github.com/YOUR_USERNAME/gitoma-bench-ladder.git
   ```

3. Create a new branch for your feature or bug fix:

   ```bash
   git checkout -b feature-name
   ```

## Development Setup

Ensure you have Python installed. The project uses Python 3.x.

Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Making Changes

- Make sure to add tests for any new functionality.
- Run tests before submitting a pull request:

   ```bash
   python -m pytest tests/
   ```

## Submitting a Pull Request

1. Push your changes to your fork:

   ```bash
   git push origin feature-name
   ```

2. Open a pull request on GitHub.
3. Provide a clear description of your changes.

## Code Style and Guidelines

- Follow the existing code style.
- Add comments for complex logic.
- Ensure your code passes all tests.

Thank you for contributing to gitoma-bench-ladder!
