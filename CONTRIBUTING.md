# Contributing to gitoma-bench-ladder

## How to contribute

1. Fork the repository and create a new branch for your feature or bugfix.
2. Make sure you have Python installed and the project dependencies installed via `pip install -r requirements.txt`.
3. Write tests for your changes in the `tests/` directory.
4. Ensure your code follows the project's coding standards and PEP8 guidelines.
5. Commit your changes with a clear, descriptive commit message.
6. Push your branch to your fork and create a pull request.

## Code of Conduct

Please follow the project's code of conduct when interacting with other contributors.
