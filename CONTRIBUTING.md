# Contributing to gitoma-bench-ladder

## How to contribute

1. Fork the repository on GitHub
2. Clone your forked repository: `git clone https://github.com/YOUR_USERNAME/gitoma-bench-ladder`
3. Create a new branch for your feature: `git checkout -b feature-name`
4. Make your changes
5. Test your changes locally
6. Commit your changes with a clear message: `git commit -m "Fix bug in X"`
7. Push your changes to your fork: `git push origin feature-name`
8. Open a pull request on GitHub

## Code of Conduct

Please follow the code of conduct outlined in the LICENSE file.

## Testing

All tests should be written using pytest. To run tests, use the command: `pytest tests/`.

## Reporting Issues

If you find a bug, please report it on the GitHub issues page with as much detail as possible.
