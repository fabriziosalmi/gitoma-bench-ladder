# Contributing to gitoma-bench-ladder

## How to Contribute

1. Fork the repository on GitHub.
2. Create a new branch for your feature or bug fix: `git checkout -b <branch-name>`.
3. Make your changes and ensure they are well-documented.
4. Run the tests to verify everything works: `pytest`.
5. Commit your changes with a clear, concise message.
6. Push your branch to your fork and create a pull request.

## Code of Conduct

Please follow the code of conduct outlined in the LICENSE file. All contributors are expected to treat each other with respect and professionalism.
