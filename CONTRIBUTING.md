# Contributing to gitoma-bench-ladder

## How to Contribute

1. Fork the repository on GitHub.
2. Create a new branch for your feature or bug fix: `git checkout -b feature-name`.
3. Make your changes and ensure they pass all existing tests.
4. Commit your changes with a clear, concise message.
5. Push your branch to your fork and create a pull request.
6. Wait for feedback and address any issues raised.

## Code of Conduct

Please follow the code of conduct outlined in the LICENSE file. All contributors are expected to treat each other with respect and professionalism.

## Testing

All new features and bug fixes should be accompanied by tests. Ensure your changes do not break existing functionality.

## Reporting Issues

If you encounter any issues or have questions, please open an issue on the GitHub repository.
