# Contribution Guidelines

## How to Contribute

1. Fork the repository and create a new branch for your feature or bug fix.
2. Make sure to test your changes locally before submitting a pull request.
3. Write clear and concise commit messages that describe the changes you're making.
4. Submit a pull request to the main repository.

Please follow the code of conduct outlined in SECURITY.md.
