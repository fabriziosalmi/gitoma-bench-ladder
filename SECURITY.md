## Vulnerability Disclosure Policy

If you discover a security vulnerability in this project, please follow these steps:

1. Verify the vulnerability exists and is not already fixed in the latest code.
2. Contact the project maintainers through our official communication channels.
3. Provide detailed information about the vulnerability, including steps to reproduce it.
4. Do not disclose or exploit the vulnerability publicly until it has been addressed.

We appreciate responsible disclosure and will respond promptly to all valid vulnerability reports.
