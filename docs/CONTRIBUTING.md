# Contributing to gitoma-bench-ladder

## How to Contribute
1. Fork the repository on GitHub.
2. Create a new branch for your feature or bug fix: `git checkout -b feature-name`
3. Make your changes and ensure all tests pass.
4. Commit your changes: `git commit -m "Your descriptive message"`
5. Push your branch to the forked repository: `git push origin feature-name`
6. Open a pull request and follow the guidelines for code reviews.

## Code of Conduct
Please follow the standard code of conduct for open-source projects. Be respectful and collaborative in all interactions.
