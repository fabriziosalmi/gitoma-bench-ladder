# Security Policy

We take security seriously. If you discover any security vulnerability in this project, please report it responsibly.

## Reporting Vulnerabilities

Please report any security concerns via [GitHub Issues](https://github.com/your-org/gitoma-bench-ladder/issues) using the appropriate security label.

## Responsible Disclosure

We follow a responsible disclosure policy. Please do not disclose vulnerabilities publicly until we have had sufficient time to address them.

## Contact

For security-related inquiries, please contact [security@example.com] (or use the appropriate channel).
