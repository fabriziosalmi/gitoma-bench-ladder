# gitoma-bench-ladder Documentation

## Getting Started

To build and view the documentation, follow these steps:

1. Install MkDocs (recommended) or Sphinx
   ```bash
   pip install mkdocs
   # or
   pip install sphinx
   ```

2. Navigate to the docs directory:
   ```bash
   cd docs
   ```

3. Build the documentation:
   ```bash
   mkdocs build
   # or
   make html
   ```

4. View the documentation in your browser:
   ```bash
   mkdocs serve
   # or
   make html && open docs/_build/html/index.html
   ```

## Documentation Tools

- [MkDocs](https://www.mkdocs.org/): Simple, flexible documentation site generator
- [Sphinx](https://www.sphinx-doc.org/): More complex, suitable for larger projects
