# gitoma-bench-ladder

This project provides a benchmarking framework for evaluating the performance of different implementations of a git-based task automation system. It includes:
- A benchmarking suite to compare different implementations
- A ladder of difficulty for testing various scenarios
- Tools for analyzing and visualizing results

## Getting Started

1. Clone the repository:
```bash
 git clone https://github.com/your-username/gitoma-bench-ladder.git
```

2. Install dependencies:
```bash
git clone https://github.com/your-username/gitoma-bench-ladder.git
 pip install -r requirements.txt
```

3. Run the benchmarks:
```bash
 python -m src.db
```
